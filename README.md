# SdmcPlugin

Plugin for Unity 5.6.6f2 that allows access to SD Card root of N3DS consoles on non-development builds. The plugin comes with a C# wrapper that exposes the necessary functions, and a sample scene that shows intended usage of the plugin.

Source code for the plugin is included.

## Features

* Reading/Writing/Deleting files with streaming support
* Creating/Deleting directories
* Checking if files/directories exist

## Plugin build requirements

* Visual Studio 2015 Professional
* VSI-CTR 5.0.4
* CTR-SDK ~11.4.0
